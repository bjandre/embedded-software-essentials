# FRDM-KL25Z

Platform specific code for the NXP / Freescale FRDM-KL25Z development
board.

## Third-party code

The following files are from the CMSIS implementation in Kinetis
Design Studio 3.2:

  * MKL25Z128xxx4\_flash.ld
  * MKL25Z4.h
  * MKL25Z4\_features.h
  * cmsis\_gcc.h
  * core\_cm0plus.h
  * core\_cmFunc.h
  * core\_cmInstr.h
  * fsl\_device\_registers.h
  * startup\_MKL25Z4.S
  * system\_MKL25Z4.c
  * system\_MKL25Z4.h
